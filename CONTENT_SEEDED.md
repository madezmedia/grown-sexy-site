## 📝 Content Guidelines (For Future Additions)

### Events
- Always include: title, subtitle, date, time, location, price, category, status
- Use "upcoming" status for new events
- Set isFeatured: true for homepage display
- Add 4-6 highlights as bullet points
- Write 500-800 words of detailed description
- Include logistics (parking, dress code, etc.)

### Artists
- Include: name, stageName, bio, genres, social links
- Add 3-5 gallery images
- Write 600-1000 words covering: background, style, achievements, philosophy
- Set featuredOnHomepage: true for homepage spotlight
- Link to upcoming performances

### Blog Posts
- Always include: title, author, category, excerpt, tags
- Categories: Events, Artist Spotlight, Community, News
- Write 500-1200 words
- Use subheadings for scanability
- Include quotes, testimonials, or data points
- End with clear call-to-action
